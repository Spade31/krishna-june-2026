### Permissions for the matrix in linux
**600** ***Strictly private***
Behavior: Only the owner can read and write to the file. Everyone else on the system is completely blocked.


**644** ***Public read private write***
Behavior: The owner can read and edit the file, while the assigned group and the rest of the world can only read it.


**755** ***Standard Executable / Public Folder***
Behavior for Files: The owner has full control; everyone else can read and run (execute) the file as a program.
